# MuscleMatrix Color Theme

## Primary Colors
- **Primary Red**: `#DC2626` (rgb(220, 38, 38)) - Main brand color
- **Secondary Red**: `#EF4444` (rgb(239, 68, 68)) - Highlights
- **Accent Red**: `#F87171` (rgb(248, 113, 113)) - Light accents
- **Dark Red**: `#B91C1C` (rgb(185, 28, 28)) - Hover states

## Usage

### Tailwind Classes (Auto-converted to Red)
- `text-emerald-400` → Red #EF4444
- `text-emerald-500` → Red #EF4444
- `text-emerald-600` → Red #DC2626
- `bg-emerald-500` → Red background
- `border-emerald-400` → Red border

### CSS Variables
```css
var(--primary-red)    /* #DC2626 */
var(--secondary-red)  /* #EF4444 */
var(--accent-red)     /* #F87171 */
var(--dark-red)       /* #B91C1C */
```
